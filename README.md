# 📖 Overview

This repository contains my personal research, notes, implementation examples, and documentation on Knowledge Graphs using Neo4j AuraDB.

The goal of this project is to understand how graph databases represent real-world entities and relationships, and how Knowledge Graphs can power intelligent applications such as AI assistants, recommendation systems, GraphRAG, fraud detection, healthcare analytics, and semantic search. 0
